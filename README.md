# Home Services Website

A complete HTML website clone based on the provided screenshots for "HS Home Services" / "Grounddown Construction".

## Files Structure

- `index.html` - Homepage with hero section, services, How It Works, Why Choose Us, and Certified sections
- `services.html` - Services page showcasing available services
- `reviews.html` - Customer reviews and testimonials page
- `areas.html` - Service areas page with location listings
- `projects.html` - Recent jobs/projects page
- `faqs.html` - Frequently Asked Questions page with accordion functionality
- `about.html` - About us page
- `contact.html` - Contact page
- `styles.css` - Complete styling matching the original design
- `script.js` - JavaScript for interactive elements (FAQ accordion, form handling, etc.)

## Features

- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Interactive Elements** - FAQ accordion, smooth scrolling, form handling
- **Color Scheme**:
  - Dark Blue (#1e3a8a, #1e40af) for headers and backgrounds
  - Orange (#f97316) for accents and CTAs
  - White and light grey for content areas
- **All Content** - Matches the original website text and structure exactly

## How to Use

1. Open `index.html` in a web browser to view the homepage
2. Navigate between pages using the header navigation
3. All pages are fully functional with proper linking

## Browser Compatibility

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
